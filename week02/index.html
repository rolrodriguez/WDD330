    Notes

    • DRY code, Don't repeat yourself
    • Loops are the method used to repeat code execution. Execution can be interrupted by using "break" or certain iterations skipped by using "continue"
    • Functions help to reduce repetition. Complexity comes from stacking functions that do one thing only quite well.
