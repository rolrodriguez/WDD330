    Notes

    • We can use the short notation for transitions, to change from:
      • transition-property: transform;
      • transition-duration: 0.2s;
      • transition-timing-function: ease-out
      • transition-delay: 50ms
      To a more convenient one:
      • transition: transform 0.2s ease-out 50ms;
    • Several transitions can be declared with the short notation by separating them with commas.
    • There is a short hand notation for animations too, it helps changing from this:
      • animation-name: appearDisappear;
      • animation-duration: 300ms;
      • animation-timing-function: ease-in;
      • animation-iteration-count: 1;
      • animation-direction: alternate;
      • animation-delay: 5s;
      To this:
      • animation: 300ms ease-in alternate 5s alternate appearDisappear;
