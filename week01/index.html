Week 01

    Exercises

    • LocalStorage

    Notes

    • LocalStorage uses key-value pairs to store data in the browser
    • We need to design sites mobile first not just responsive, responsive normally means adding media queries to an already desktop site, to make it more mobile friendly. Mobile first means that we first design it for mobile and then we think about how will it look on bigger screens
    • Mobile first design makes us focus on simplicity and what is truly needed
    • When designing a site for mobile, we need to place menus closer to the user, having menus at the bottom can help users to make your app/site more usable

    Questions:

    • How is LocalStorage implemented? Is it a file created per site or is it a common file for all sites?
    • How safe is LocalStorage? Can it be downloaded by the user?
