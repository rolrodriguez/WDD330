    Exercises

    • Forms, OOP and Modern JS
    • Tic, tac, toe game

    Notes

    • forms["form name"].elements returns a collection of the elements in the form
    • Use event.preventDefault() to stop a form from being submitted
    • In real-life we should use validation in both the front end and the back end.
    • If we want to add additional functionality to an existing class we have to options:
      1. To add the additional property to the prototype
      2. Create a new class inheriting from the class we want to use as a basis
    • We can create a new instance of an already created object by using: let myObject = Object.create(NameOfObject)
    • If a method returns this, its methods can be chained together to form a sequence of method calls that are called one after the other.
    • By using CDNs (Content Delivery Networks) for our libraries we save downloading them again for the user if the same are used in other sites, like JQuery, Angular, and others.
    • A code transpiler allows us to write code in the latest version of ECMAScript, using all the latest features, then convert it into an older variant of JavaScript that is compatible with most browsers. Babel is a famous code transpiler.
